”爬，都可以爬“这个程序是用来完成第一个任务的，通过post请求获取主页关键字搜索发送的json，然后用正则表达式筛选出来关键信息，然后每一组数据放在一个字典里，所有的字典都放进一个集合里，这样就可以获取关键字搜素的其中一页书单组成的列表。

输入关键字 空格 想看的页数。

输出书单。

”爬馆藏“的基本原理和第一个相同，只不过端口是在某一本书的详情网页扫描出来的，用的请求方式为get，由于太过疲倦，忘了给输入提示，使用时请直接输入书的recordid

输入recordid

输出书的id，条形码，馆藏地，是否在架元组组成的列表
